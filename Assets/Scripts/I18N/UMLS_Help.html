Unity Multiple Language Support
Add support for multiple languages to your Unity projects by storing strings with multiple translations in one or more XML files and referencing those strings with a common string identifier.  Simply place the Lang.js file under your project's Assets folder or any subfolder thereof then create an XML file containing all necessary strings, along with their translations, and save it to your Assets folder as-well.  Your XML document should have the following format:

<?xml version="1.0" encoding="utf-8"?>
<languages>
    <English>
         <string name="app_name">Some Game</string>
         <string name="score_text">Score: </string>
    </English>

    <French>
        <string name="app_name">Certains Jeux</string>
        <string name="score_text">Score: </string>
    </French>

    <German>
        <string name="app_name">Einige Spiele</string>
        <string name="score_text">Ergebnis: </string>
    </German>
</languages>

Each language is referenced by the names of the Root Node's children, in this example languages would be referenced with "English" "French" or "German".  Within each language node you will list out your strings each with an identifier that is unique within it's own language node, but identical to the same string in another language.  In this example strings are referenced with the identifiers "app_name" and "score_text".

Note:
Using a single XML file is probably fine for smaller games, but you may find that RAM usage can be decreased on larger games by using multiple XML files such as LVL_01.xml, LVL_02.xml, etc...


Using the Lang Class

The Lang class can be used in conjunction with locally stored XML resources or XML resources stored on the internet such as would be the case with web-player games.

Constructor:
Lang( path : String, language : String, web : boolean )
path:  String.  File path to the XML resource if locally stored, XML resource text if web based
language:  String.  Language to be used
web:  Boolean.  False if XML resource is stored locally otherwise true

Usage (JavaScript, local XML resource):
import System.IO;

var langClass : Lang;

function Start () {
    langClass = new Lang(Path.Combine(Application.dataPath, "lang.xml"), "English", false);
}

Usage (JavaScript, web XML resource):
import System.IO;

var langClass : Lang;

function Start () {
    var wwwXML : WWW = new WWW("http://www.exampleURL.com/lang.xml");
    yield wwwXML;

    langClass = new Lang(wwwXML.text, "English", true);
}

Methods:
setLanguage:
setLanguage( path : String, language : String )
path:  String.  File path to the XML resource
language:  String.  Language to be used
Use the setLanguage function to switch the current language selection after having initialized the class such as from an options screen.  This function is automatically called when the Lang class is initialized.  Used with local XML resources.

Usage (JavaScript, local XML resource):
import System.IO;

var langClass : Lang;
var currentLang : String = "English";

function Start () {
    langClass = new Lang(Path.Combine(Application.dataPath, "lang.xml"), currentLang, false);
}

function OnGUI () {
    if (GUI.Button(Rect(0, 450 - 24, 800, 24), "Language:  " + currentLang)) {
        if (currentLang == "English") {
            currentLang = "French";
        } else if (currentLang == "French") {
            currentLang = "German";
        } else {
            currentLang = "English";
        }
       
        langClass.setLanguage(Path.Combine(Application.dataPath, "lang.xml"), currentLang);
    }
}

setLanguageWeb:
setLanguageWeb( xmlText : String, language : String )
path:  String.  XML resource as text
language:  String.  Language to be used
Use the setLanguageWeb function to switch the current language selection after having initialized the class such as from an options screen.  This function is automatically called when the Lang class is initialized.  Used with web based XML resources.

Usage (JavaScript, web XML resource):
import System.IO;

var langClass : Lang;
var currentLang : String = "English";

function Start () {
    var wwwXML : WWW = new WWW("http://www.exampleURL.com/lang.xml");
    yield wwwXML;

    langClass = new Lang(wwwXML.text, "English", true);
}

function OnGUI () {
    if (GUI.Button(Rect(0, 450 - 24, 800, 24), "Language:  " + currentLang)) {
        if (currentLang == "English") {
            currentLang = "French";
        } else if (currentLang == "French") {
            currentLang = "German";
        } else {
            currentLang = "English";
        }
       
        var wwwXML : WWW = new WWW("http://www.exampleURL.com/lang.xml");
        yield wwwXML;       

        langClass.setLanguage(wwwXML.text, currentLang);
    }
}

getString:
getString( name : String ) : String
name:  String.  Unique string identifier
Returns:  String associated with the XML "name" identifier in the currently set language.
Use the getString function to access a desired string in the currently selected language.

Usage (JavaScript):
import System.IO;

var langClass : Lang;
var currentLang : String = "English";

var Score : int = 0;
var scoreText : String;

function Start () {
    langClass = new Lang(Path.Combine(Application.dataPath, "lang.xml"), currentLang, false);
    scoreText = langClass.getString("score_text") + "0";
}

function scoreChange ( newScore : int ) {
    Score = newScore;
    scoreText = langClass.getString("score_text") + newScore.ToString();
}

function OnGUI () {
    GUI.Label(Rect(0, 0, 192, 450), langClass.getString("app_name") + "\n" + scoreText);
}
